# Yogi

The pinnacle of finance for your communities.

[Invite the bot](https://discord.gg/TxWpjjMed6) | [Join `theluqmn's garage`](https://discord.gg/TxWpjjMed6)

> [!NOTE]
> Yogi is still under active development.

## Features

- **No risks attached**. Fully simulated, with real-time, real-world data.
- **It is complete**. Access, trade, and exchange hundreds of tokens and currencies worldwide.
- **Beyond individuals**. Transfer assets between fellow friends and community members.

## Technical Overview

Yogi is written in Python 3.12, using the [Pycord](https://pycord.dev) library (a fork of discord.py). The database utilised is SQlite, integrated via the built-in `sqlite3` Python library.

Yogi utilises the following APIs:

1. [Coinbase](https://coinbase.com) - empowers most of Yogi's currency features.
